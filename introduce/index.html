# 搜尋系統

搜尋系統分成兩大類

1.關鍵字搜尋

卡片名稱只要有包含關鍵字都會顯示出來

2.屬性搜尋

1.精準搜尋 > 只會列出完全匹配的卡片

2.模糊搜尋 > 列出所有包含所選取屬性的卡片

判斷為精準搜尋或模糊搜尋的條件如下:

1. 科目:
    只勾選 一大類 或 不勾
2. 階級:
    只勾選 一大類 或 不勾
3. 答案:
    只勾選 一個 或 不勾

(上述每一項都要符合才會啟動精準搜尋)
(生物 理化 地科 為一大類)
(歷史 地理 公民 為一大類)

1. 科目:
    勾選超過一大類
2. 階級:
    勾選超過一大類
3. 答案:
    勾選超過一大類

(上述任何一項只要符合就會啟動模糊搜尋)
(生物 理化 地科 為一大類)
(歷史 地理 公民 為一大類)

# 登入有哪些好處

登入後將新增以下功能
• 您會有一個獨立的個人資料系統
• 您可以對上傳過的檔案進行管理
• 您上傳的檔案將會有您的個人資訊，不在是匿名
• 非匿名上傳的檔案不會被列為危險檔案
