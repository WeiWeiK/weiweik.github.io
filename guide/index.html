# 關鍵字搜尋

>搜尋系統如何運作

# 上傳檔案

注意 !

題目檔大小限制為 6MB
答案檔大小限制為 6MB
答案檔的圖檔大小限制為 9MB
1.首頁點擊上傳按鈕 (按鈕如下)
2.進入頁面後選擇:

1.點擊 選取檔案 從檔案總管上傳

或

2.將檔案拖曳至框框裡放開

注意 !

題目檔類型只支援  .pdf .doc .docx
3.點擊 下一步

要取消點擊 移除檔案 即可

4.完成以下步驟:

提示 !

底部有即時動態預覽圖

1.檔案名稱:

只接受 中文, 英文, 數字, -

2.階級:

試卷適合的階級 只能勾選一項

3.科目:

試卷所屬的科目 特定項目可二至三項

4.有無答案:

此試卷是否有 答案

如果有請勾選

反之則勾選

勾選有答案後如需額外上傳答案檔請勾選

反之則勾選

5.以上完成後上傳按鈕會出現

點擊上傳後耐心等待即可上傳

注意 !

上傳中勿關閉頁面，否則會上傳失敗

# 註冊帳號

>登入有哪些好處
1.點擊登入按鈕 (按鈕如下)
登入
2.點擊右下角的註冊
註冊
3.輸入資料

1.Gmail: 輸入Gmail信箱

2.使用者名稱: 您的帳戶名稱

注意 !

帳戶名稱只接受 1 個字元以上 5 個字元以下的 中文 或 英文

3.密碼: 您想設定的密碼

注意 !

密碼格式請至少包含一個 數字 及一個 英文字母 字元長度大於 7

4.再次輸入密碼: 再次輸入密碼

4.資料確認無誤後請按註冊等待接收驗證碼

請去Gmail信箱確認

注意 !

如果Gmail信箱無驗證碼請檢查 垃圾郵件
5.輸入7位數的驗證碼
6.驗證碼確認無誤後請按發送即成功

# 登入帳號

>登入有哪些好處
1.點擊登入按鈕 (按鈕如下)
登入
2.輸入使用者帳密

1.帳號: 當初註冊時使用的Gmail

2.密碼: 當初註冊時設定的密碼

3.資料確認無誤後請按登入即成功

# 忘記密碼

警告 !

只有註冊過的用戶且帳號 未刪除 才有這項功能
1.點擊登入按鈕 (按鈕如下)
登入
2.點擊左下角的忘記密碼
忘記密碼
3.輸入使用者帳號

1.帳號: 當初註冊時使用的Gmail

4.帳號確認無誤後請按發送
5.前往Gmail信箱 點擊收到的郵件裡面的連結

注意 !

如果Gmail信箱無新郵件請檢查 垃圾郵件
6.進入連結後依照提示即可完成更改動作
